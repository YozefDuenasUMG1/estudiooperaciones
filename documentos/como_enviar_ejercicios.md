# Cómo Enviar Ejercicios para Análisis

## Formato Recomendado para Ejercicios

### Información Mínima Requerida:
1. **Contexto del problema** (industria, situación, etc.)
2. **Datos numéricos completos** (costos, capacidades, demandas, etc.)
3. **Pregunta específica** (¿qué se necesita calcular o determinar?)
4. **Restricciones o limitaciones** si las hay

### Ejemplo de Formato Ideal:

```markdown
## Contexto
Una empresa de [industria] necesita [descripción del problema]...

## Datos
- Variable 1: [valor] [unidades]
- Variable 2: [valor] [unidades]
- Costos: [tabla o lista de costos]
- Capacidades: [límites de recursos]

## Pregunta
¿Cuál es la [solución específica que se busca]?
```

## Tipos de Ejercicios Que Puedo Analizar

### 1. Programación Lineal
- Problemas de producción
- Mezcla de productos
- Asignación de recursos
- Planificación de dietas
- Problemas de inversión

### 2. Problemas de Transporte
- Distribución de productos
- Asignación de plantas a mercados
- Localización de almacenes
- Rutas de entrega

### 3. Problemas de Asignación
- Asignación de trabajadores a tareas
- Asignación de máquinas a trabajos
- Programación de horarios
- Emparejamiento óptimo

### 4. Teoría de Colas
- Sistemas de espera en bancos
- Centros de servicio al cliente
- Sistemas de producción
- Análisis de tráfico

### 5. Gestión de Inventarios
- Cálculo de lotes económicos
- Inventarios con descuentos
- Sistemas de revisión continua
- Análisis ABC

### 6. Análisis de Decisiones
- Árboles de decisión
- Análisis de riesgo
- Decisiones bajo incertidumbre
- Teoría de juegos básica

## Proceso de Análisis

### Cuando Envíes un Ejercicio:

1. **Análisis Automático**
   - El sistema identificará el tipo de problema
   - Sugerirá métodos de solución apropiados
   - Recomendará la plantilla a usar

2. **Formulación del Modelo**
   - Se definirán las variables de decisión
   - Se establecerá la función objetivo
   - Se identificarán todas las restricciones

3. **Solución Paso a Paso**
   - Se aplicará el método más apropiado
   - Se mostrarán todos los cálculos
   - Se verificará la factibilidad

4. **Interpretación de Resultados**
   - Explicación de la solución óptima
   - Análisis de sensibilidad (si aplica)
   - Recomendaciones prácticas

## Consejos para Mejores Resultados

### ✅ Hacer:
- Incluir todos los datos numéricos
- Especificar unidades de medida
- Describir el contexto claramente
- Indicar exactamente qué se necesita calcular

### ❌ Evitar:
- Ejercicios incompletos sin datos
- Preguntas vagas o ambiguas
- Omitir restricciones importantes
- No especificar el objetivo (maximizar/minimizar)

## Ejemplo de Ejercicio Bien Formulado

```
**Contexto:** Una fábrica de muebles produce sillas y mesas.

**Datos de Producción:**
- Silla: 3 horas de carpintería, 1 hora de acabado, ganancia $40
- Mesa: 2 horas de carpintería, 2 horas de acabado, ganancia $60

**Recursos Disponibles:**
- Carpintería: 120 horas/semana
- Acabado: 80 horas/semana

**Pregunta:** ¿Cuántas sillas y mesas debe producir semanalmente para maximizar la ganancia?
```

Este formato permite un análisis completo y una solución precisa.
