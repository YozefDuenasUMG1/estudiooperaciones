# Estudio de Operaciones

Este repositorio contiene análisis y soluciones de ejercicios de Investigación de Operaciones.

## Estructura del Repositorio

- `ejercicios/` - Ejercicios organizados por tema
- `soluciones/` - Soluciones detalladas paso a paso
- `plantillas/` - Plantillas para diferentes tipos de problemas
- `herramientas/` - Utilidades y herramientas de análisis
- `documentos/` - Documentación teórica y referencias

## Tipos de Ejercicios Soportados

1. **Programación Lineal**
   - Método Simplex
   - Método Gráfico
   - Dualidad

2. **Problemas de Transporte**
   - Método de la Esquina Noroeste
   - Método del Costo Mínimo
   - Método de Vogel

3. **Problemas de Asignación**
   - Método Húngaro

4. **Teoría de Colas**
   - Modelos M/M/1, M/M/s
   - Análisis de sistemas de espera

5. **Teoría de Inventarios**
   - Modelo EOQ
   - Inventarios con demanda probabilística

6. **Análisis de Decisiones**
   - Árboles de decisión
   - Criterios de decisión bajo incertidumbre

## Cómo Usar Este Repositorio

1. Coloca tu ejercicio en la carpeta `ejercicios/` con una descripción clara
2. El análisis y solución se generará en `soluciones/`
3. Consulta las plantillas disponibles para el tipo de problema correspondiente
